# Yahoo Finance Outputs — Schema Guide

This repository contains Yahoo Finance–derived datasets generated by `yahoo_crawler.py`. This README documents the schema for each Yahoo output file and how the files relate to each other.

## Overview

**Primary relationships**
- `index_symbol_map.csv` maps NASDAQ index identifiers to Yahoo symbols.
- `yahoo_timeseries.csv` is the combined time‑series table (price + technical indicators) keyed by `(index_id, date)`.
- `yahoo_indicators.csv` contains point‑in‑time fundamental/valuation/analyst/ESG indicators keyed by `index_id` and `as_of_date`.
- `yahoo_options.csv` contains options chain data when available for the Yahoo symbol.
- `yahoo_crawl_log.csv` records request outcomes, including the crumb used for authenticated requests.

> Note: `index_master.csv` is **optional** and may be missing in this folder. When present, it provides the base index list used by the crawler.

---

## `index_symbol_map.csv`
**Purpose**: Maps NASDAQ index identifiers to Yahoo Finance symbols (best match from Yahoo search).

**Columns**
- `index_id`: Internal ID for the index (e.g., `^IXIC`).
- `yahoo_symbol`: Resolved Yahoo symbol (e.g., `^IXIC`).
- `confidence`: Match confidence score (0–1) based on symbol/name/quoteType matching.
- `quoteType`: Yahoo quote type (expected `INDEX`).
- `matched_name`: Name returned by Yahoo for the matched symbol.
- `source_url`: Yahoo search URL used for the match.

**Notes**
- If no Yahoo match is found, `yahoo_symbol` and related fields may be empty.

---

## `yahoo_timeseries.csv`
**Purpose**: Combined time‑series data (prices + technical indicators). One row per `(index_id, date)`.

**Columns**
- `index_id`: Internal index identifier.
- `date`: UTC date in `YYYY-MM-DD`.
- `level_open`: Daily open price.
- `level_high`: Daily high price.
- `level_low`: Daily low price.
- `level_close`: Daily close price.
- `total_return_level`: Reserved for total‑return series (empty for Yahoo chart data).
- `source_url`: Yahoo chart endpoint URL used for price data.
- `source`: Data source label (e.g., `yahoo`).
- `ma_50`: 50‑day simple moving average of close.
- `ma_200`: 200‑day simple moving average of close.
- `rsi_14`: 14‑day Relative Strength Index.
- `macd`: MACD (EMA12 − EMA26).
- `macd_signal`: MACD signal line (EMA9 of MACD).
- `macd_hist`: MACD histogram (`macd` − `macd_signal`).
- `bb_upper`: Bollinger Band upper (20‑day SMA + 2σ).
- `bb_mid`: Bollinger Band middle (20‑day SMA).
- `bb_lower`: Bollinger Band lower (20‑day SMA − 2σ).
- `stoch_k`: Stochastic %K (14‑period).
- `stoch_d`: Stochastic %D (3‑period SMA of %K).
- `technical_source_url`: Yahoo chart endpoint URL used for technicals (same source as price data).

**Notes**
- Indicators may be empty for early periods without enough history (e.g., MA‑200).

## `yahoo_indicators.csv`
**Purpose**: Point‑in‑time fundamental, valuation, analyst, ESG, and profile indicators from Yahoo `quoteSummary` modules.

**Columns**
- `index_id`: Internal index identifier.
- `yahoo_symbol`: Yahoo symbol the indicators were pulled for.
- `indicator_group`: Logical grouping (e.g., `price_trading`, `valuation`, `profitability`, `balance_sheet`, `income_cashflow`, `earnings`, `dividends`, `analyst`, `additional`).
- `indicator_name`: Specific metric name (e.g., `trailingPE`, `marketCap`, `dividendYield`).
- `indicator_value`: Metric value (raw numeric or string). Empty if not available for the symbol.
- `as_of_date`: Date the metric was retrieved (UTC date).
- `source_url`: Yahoo quoteSummary URL used for retrieval.

**Notes**
- Many fundamentals are **not available for indices**, so values may be empty.

---

## `yahoo_crawl_log.csv`
**Purpose**: Request‑level crawl log for Yahoo endpoints.

**Columns**
- `timestamp`: UTC timestamp of request.
- `url`: URL requested.
- `status`: HTTP status or error label.
- `reason`: Short reason or retry annotation.
- `crumb`: Yahoo crumb used (if applicable).

**Notes**
- The `crumb` value is included to reproduce Yahoo API calls.

---

## `index_master.csv` (Optional)
**Purpose**: Base list of indices used by the crawler (if present).

**Typical Columns**
- `index_id`, `symbol`, `name`, `currency`, `category_group`, `category`, `exchange`, `source_url`
- Yahoo‑augmented columns (if created by the crawler):
  - `yahoo_symbol`, `yahoo_exchange`, `yahoo_currency`, `yahoo_instrumentType`

**Notes**
- This file may not exist if it has been removed during cleanup.
